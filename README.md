# Ternary Capital — Prompts

Prompt library for cognitive financial agents — research, portfolio management, risk, and analysis workflows.
Part of Ternary's open-source stack for agentic financial intelligence.

## What this is

A curated collection of prompts designed to guide agents through structured financial reasoning. Each prompt is built around Ternary's agent properties: grounded context, persistent memory, goal-directed planning, and self-correction.

## Categories

- **Agent** — task delegation prompts for autonomous financial workflows
- **Analysis** — document analysis: 10-Ks, earnings calls, filings
- **Portfolio** — position management, concentration, and rebalancing guidance
- **Research** — deep-dive investigation prompts for equity and macro research
- **Risk management** — scenario analysis, downside mapping, and preservation protocols
- **Knowledge graph** — entity extraction and structured reasoning over relationships
- **Summarisation** — condensing long-form financial documents into actionable insight
- **Trading** — experimental prompts for autonomous trade reasoning (not for live use)

## Open Source

Part of [github.com/ternary-ai](https://github.com/ternary-ai) — open-source agents, models, synthetic data, prompt engineering resources, and scaffolding maintained for the community.

## Disclaimer

All prompts are provided as-is. Output quality depends on the underlying model and context. Ternary Capital accepts no liability for outcomes arising from use of this material. Trading prompts are experimental and must not be used for live execution.
